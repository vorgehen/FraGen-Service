# FraGen Service
## Quick Start
1. Configure FastAPI in fraGen_service_config.toml
2. Add OpenAi Key to fraGen_service_config.toml
3. Start: python fraGen_service_fastapi.py
4. Use: http://127.0.0.1:5000/docs
